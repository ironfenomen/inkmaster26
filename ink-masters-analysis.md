### 2.3. SEO-стратегия по этапам

#### ЭТАП 1 (Месяц 1–2): Фундамент

**Техническое SEO:**
- Сайт на Next.js 14 (SSR/SSG для Яндекса) — ты уже работаешь с фреймворком
- Хостинг на VPS (Beget, как у nashdomvashdom) — SSR требует сервер
- SSL-сертификат (Let's Encrypt)
- Robots.txt + Sitemap.xml
- Яндекс.Вебмастер — подтверждение, регион «Ставрополь»
- Яндекс.Метрика — цели: клик по телефону, отправка формы, клик VK или Telegram
- Скорость загрузки: Core Web Vitals в зелёной зоне
- Адаптивный мобильный дизайн (60%+ трафика — мобильные)
- Микроразметка Schema.org: LocalBusiness, FAQPage, Review, ImageGallery

**Привязка к региону:**
- В Яндекс.Вебмастере: регион = Ставрополь
- NAP (Name, Address, Phone) одинаковые на сайте, картах, соцсетях
- Адрес в footer + страница контактов с Яндекс.Картой
- Привязка Яндекс.Бизнес к сайту (ссылка с карточки на сайт)

**Контент на старте:**
- Главная: H1 «Тату-студия Ink Masters в Ставрополе», USP, CTA
- 5 страниц портфолио (основные стили)
- 3 страницы услуг
- Страница цен (с примерами и калькулятором)
- Страница о мастере (биография, опыт, подход)
- Страница FAQ (15–20 вопросов)
- 3 статьи в блог

### 2.5. Оптимизация для ИИ-поиска

**Что такое ИИ-поиск и почему это важно:**

Яндекс.Нейро, AI Overview Google, Perplexity, ChatGPT с поиском — всё это ИИ-поисковики, которые выбирают ответ из сайтов. Чтобы попасть в ответ ИИ:

1. **Прямые ответы на вопросы.** Каждая статья начинается с чёткого ответа на вопрос из заголовка (первый абзац = ready-made snippet).

2. **Структура «вопрос → ответ».** FAQ-разделы на каждой странице. Яндекс.Нейро берёт оттуда ответы.

3. **E-E-A-T сигналы (Experience, Expertise, Authority, Trust):**
   - Автор статей = мастер Денис (реальный человек, профиль на сайте)
   - Портфолио с реальными работами = доказательство экспертности
   - Отзывы клиентов = траст
   - Стаж работы, количество выполненных работ = авторитет

4. **Микроразметка Schema.org:**
   - FAQPage — для FAQ
   - LocalBusiness — для карточки студии
   - Review — для отзывов
   - HowTo — для инструкций по уходу
   - Article — для блога

5. **Длинный экспертный контент.** ИИ-поисковики предпочитают страницы с развёрнутым, уникальным контентом (2000–5000 слов) перед короткими лендингами.

1. **Пустое поле.** Ни один конкурент в Ставрополе не имеет профессионально оптимизированного сайта. Nazarov — на Tilda (SEO-потолок), Кольт — устаревший CMS, остальные — без сайтов вообще.

2. **Техническое преимущество.** Next.js 14 с SSG/SSR — это уровень, которого нет ни у одного конкурента. Яндекс любит быстрые, структурированные сайты.

3. **Контентное преимущество.** Регулярный блог + портфолио с фильтрами + FAQ = контентная масса, которой нет у конкурентов.

4. **Мастер с историей.** Денис — реальный мастер с портфолио, отзывами, активными соцсетями. Это E-E-A-T, который невозможно подделать.

5. **Интеграция с экосистемой.** Сайт + Яндекс.Бизнес + 2GIS + VK + Instagram = единая цифровая экосистема, которая усиливает каждый канал.

### Главные риски:

1. **Контент нужно создавать регулярно.** Без постоянного наполнения блога позиции будут расти медленно.
2. **Отзывы на картах — критично.** Без активной работы с отзывами карточки будут проигрывать конкурентам (Гудвин: 123 отзыва).
3. **Фото-контент.** Нужны качественные фото работ, процесса, студии. Стоковые фото = смерть для тату-сайта.
